# 📚 本科通信工程核心课程学习笔记

## 简介
这里存放了我本科期间关于通信工程专业核心课程的学习笔记。内容涵盖了通信原理、信号与系统、信息论等基础学科。
笔记内容主要基于课堂讲解与个人复习总结，旨在帮助大家复习核心概念和公式推导。

## 📂 笔记列表
* **通信原理**：涵盖模拟/数字调制、信道模型等。
* **信号与系统**：时域/频域分析、拉普拉斯变换、Z变换等。
* **信息论**：熵、互信息、信道容量、编码理论等。
* **通信知识汇总**：多个专业课的精品复习知识点梳理和更深的认知。
